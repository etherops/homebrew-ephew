# homebrew-ephew

Homebrew tap for [ephew](https://github.com/etherops/ephew) — a local verbosity-toggle proxy for the Anthropic API.

## Install

```bash
brew tap etherops/ephew
brew install ephew
ephew
```

That's it. `ephew` is now on your `PATH` and the menu-bar daemon will start when you run it.

## Update

```bash
brew update
brew upgrade ephew
```

## Uninstall

```bash
brew uninstall ephew
brew untap etherops/ephew
```
